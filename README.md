# Forge

Local-first character sorting app built with React, Vite, Tailwind, and React Query.

## Getting started

1. Install dependencies: `npm install`
2. Run the app: `npm run dev`

## Data storage

This app is independent of Base44. Data is stored in the browser's localStorage under the key `forge_data_v1`.

## Build

- `npm run build`
- `npm run preview`
